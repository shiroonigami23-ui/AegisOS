# Security Policy

## Reporting a Vulnerability

Please do not open public issues for active vulnerabilities.

Report privately via GitHub Security Advisories for this repository.

Include:

- Affected component and version/commit.
- Reproduction steps or proof of concept (PoC).
- Potential impact.
- Suggested mitigations, if known.

## Supported Versions

This project is pre-alpha. Security fixes target the latest `main` branch.
